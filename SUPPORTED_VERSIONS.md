# Supported-Version Policy

Player support is provided only for the current public release shown on the
official itch.io or CurseForge page and its listed game version/dependencies.

Before opening an issue, install a fresh official release, confirm the game is
on a supported version, and remove outdated copies of the mod. Source builds,
GitHub ZIP downloads, experimental branches, modified files, and third-party
reuploads are unsupported.

Bug reports should include the official release version, game version, installed
dependencies, other relevant mods, and clear reproduction steps.
